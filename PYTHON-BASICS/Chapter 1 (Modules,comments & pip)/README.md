### 1. Modules
Modules are Python files that contain reusable code.

- **Built-in Modules**: Available by default (e.g., `os`, `math`, `random`)
- **External Modules**: Installed using pip (e.g., `numpy`, `pandas`)

Example:
```python
import math
print(math.sqrt(16))
````

---

### 2. Comments

Comments are ignored by the interpreter and are used to improve code clarity.

* **Single-line**: `# comment`
* **Multi-line / Docstrings**: `''' comment '''` or `""" comment """`

Use comments to explain *why* logic exists, not what obvious code does.
